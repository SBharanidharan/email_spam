Evaluation Metrics Guide

What do these Machine Learning scores mean?

Accuracy: Overall correct predictions / total predictions. Excellent baseline, but can be misleading for imbalanced datasets.
Precision: True Spams / (True Spams + False Spams). Out of all emails classified as Spam, how many were actually Spam? Critical to avoid blocking important emails!
Recall: True Spams / (True Spams + Missed Spams). Out of all actual Spam emails, how many did the model block?
F1-Score: Harmonic mean of Precision and Recall. Balances both metrics into a single representative score.
